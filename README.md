## Prerequisites

* Linux
* PHP >=8.1
* Composer
* Make
* PostgeSQL

## Setup database
1. create empty PostgreSql database
2. add database URL to enviroment variable DATABASE_URL

## Setup project
```bash
git https://github.com/vsev92/php-project-48.git
cd  php-project-48
make install
make start
```
then web service is avaible on 0.0.0.0:8000
